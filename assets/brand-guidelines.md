# Diretrizes de Identidade Visual - Pechinchoso

## Paleta de Cores
- **Primária**: #FF6B35 (Laranja vibrante - representa energia e oportunidades)
- **Secundária**: #2EC4B6 (Verde-azulado - representa economia e sustentabilidade)
- **Acento**: #FF9F1C (Amarelo dourado - representa descontos e valor)
- **Neutro**: #F7F7F7 (Cinza claro - para textos e fundos)
- **Escuro**: #2C2C2C (Preto suave - para textos principais)

## Tipografia
- **Título Principal**: Montserrat Bold
- **Subtítulos**: Montserrat SemiBold
- **Corpo**: Montserrat Regular
- **Destaque**: Montserrat Medium

## Logo
O logo combina:
- Um ícone de sacola de compras estilizada
- A palavra "Pechinchoso" em fonte personalizada
- Elementos que sugerem economia e descontos

## Aplicação
- Usar cores primárias para CTAs e elementos importantes
- Manter contraste adequado para acessibilidade
- Usar gradientes sutis para elementos premium
- Manter consistência visual em todas as plataformas
